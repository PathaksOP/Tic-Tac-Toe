# Tic-Tac-Toe

A simple web-based Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features

* Two-player gameplay
* Win & draw detection
* Reset / New Game option

## Tech Stack

* HTML
* CSS
* JavaScript
